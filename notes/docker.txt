도커 - 데이터 또는 프로그램을 격리시키는 기능

docker version
docker -ls
docker ps -a 모든 컨테이너 표시
docker ps 현재 실행 중인 컨테이너 표시
docker run (docker container run) 하위 3가지 기능 실행
  - pull(이미지내려받기), create(컨테이너 생성), start(컨테이너 실행)
  옵션 -d, -p, -e, -dit 등등
